# teluxn

teluxn is an [uxn](https://wiki.xxiivv.com/site/uxn.html) emulator for the monome teletype.

| addr | device | 0x00   | 0x08 |
| ---- | ------ | ------ | ---- |
| 0x30 | gpio   | vector |      |
|      |        | tr out |      |
| 0x80 | ctrl   | vector |      |
|      |        | button |      |
|      |        | key    |      |
